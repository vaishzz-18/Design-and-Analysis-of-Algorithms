Urban Mobility Optimization

Business Cases Identified

Problem Title: Urban Mobility Optimization
SI. No. Business Case Identified Description Owner Probable Tools
1 Route and Vehicle Management Manage details of routes and vehicles, including distance, fare, and type of transport. Team A Arrays, Structures
2 Pollution Monitoring Monitor pollution levels associated with different vehicle types. Team B Linked Lists
3 Resort Recommendation Recommend nearby resorts based on user's destination using a binary search tree. Team C Binary Search Tree
4 Payment Processing Manage and process payments at the end of journeys. Team D Stack
5 Feedback Collection and Display Collect user feedback and display it using queue and array structures. Team E Queue, Arrays
6 Path Optimization Optimize paths using algorithms like Floyd-Warshall, Dijkstra, Prim's, and Kruskal's. Team F Graph Algorithms, Structures
7 Vehicle Cost Sorting Sort vehicles based on cost, distance, or seats using sorting algorithms like bubble, selection, and insertion sort. Team G Bubble Sort, Selection Sort, Insertion Sort
8 Vehicle Pollution Search Search for vehicles based on pollution levels using the Rabin-Karp algorithm. Team H Rabin-Karp String Search
9 Resort Search Search for resorts by rating using binary search tree traversal. Team I Binary Search Tree
10 Payment Calculation Calculate total journey cost using data structures like Fenwick Tree for installment management. Team J Fenwick Tree, Stack
